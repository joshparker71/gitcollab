Wave Function Collapse Algorithm

Learn about this fascinating procedural generation technique!

What is the Wave Function Collapse Algorithm?

The Wave Function Collapse (WFC) algorithm is a procedural content generation algorithm, popularized for its use in generating textures, levels, and entire worlds in video games and simulations. Inspired by quantum mechanics, the algorithm takes a set of input patterns and generates a cohesive output based on constraints.

How Does it Work?

The WFC algorithm starts with a grid of cells, each with a potential set of states. At each step, the algorithm selects a cell with the least entropy (the fewest possible states) and "collapses" it to a specific state based on neighboring constraints. This process continues until every cell is resolved into a specific state.

Applications of WFC

  • Level generation in video games
  • Texture and pattern generation
  • 3D structure and architecture modeling
  • Map and environment creation for simulations
